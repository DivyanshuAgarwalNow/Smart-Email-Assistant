Smart Email Assistant

This project is a Smart Email Assistant that helps generate email replies using OpenAI, a Spring Boot backend, a React frontend, and a Chrome extension integrated directly into Gmail.

The goal of this project was to build something practical that shows how AI can be embedded into everyday workflows like email communication.

What this project does

Adds an AI Reply button inside Gmail using a Chrome extension

Sends the email content to a Spring Boot backend

Uses OpenAI to generate a reply based on the selected tone

Returns a ready-to-edit email response

Also provides a React web interface for generating replies outside Gmail
